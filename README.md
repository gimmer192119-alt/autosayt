# Бот для автоматической регистрации аккаунтов Artillect

## Описание
Бот автоматически создает аккаунты на сайте artillect.pro через реферальную ссылку, используя временную почту.

## Возможности
- ✅ Автоматическая генерация временной почты через tempamail.com
- ✅ Генерация случайного имени и пароля (минимум 9 символов)
- ✅ Регистрация с подтверждением публичной оферты
- ✅ Автоматическое получение и обработка письма с подтверждением
- ✅ Извлечение ссылки подтверждения из письма
- ✅ Автоматическое подтверждение email
- ✅ Уведомления в Telegram о статусе регистрации
- ✅ Режим авто-регистрации по расписанию

## Требования
- Python 3.8+
- Библиотеки: requests, python-telegram-bot

## Установка зависимостей
```bash
pip install requests python-telegram-bot
```

## Настройка Telegram бота

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Узнайте свой Chat ID (можно через бота @userinfobot)

## Запуск

### Разовая регистрация
```bash
python main.py <BOT_TOKEN> <CHAT_ID>
```

Пример:
```bash
python main.py 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz 987654321
```

### Команды бота

После запуска бота используйте следующие команды в Telegram:

- `/start` - Показать справку по командам
- `/register` - Создать один аккаунт
- `/start_auto <минуты>` - Запустить авто-регистрацию каждые N минут
  - Пример: `/start_auto 5` - регистрация каждые 5 минут
- `/stop_auto` - Остановить авто-регистрацию
- `/status` - Показать текущий статус

## Пример работы

1. Запускаете бота: `python main.py YOUR_TOKEN YOUR_CHAT_ID`
2. Отправляете команду `/register` или `/start_auto 1`
3. Бот:
   - Создает временную почту
   - Генерирует имя и пароль
   - Регистрирует аккаунт с реферальным кодом B3B7C1C4
   - Ждет письмо с подтверждением
   - Извлекает ссылку из письма
   - Подтверждает email
   - Отправляет вам данные аккаунта

## Структура ответа

При успешной регистрации вы получите сообщение:

```
🎉 Аккаунт успешно создан и подтвержден!

👤 Имя: Alex Smith
📧 Email: random123@tempamail.com
🔒 Пароль: Xy9!kLm2@pQ
🔗 Реферальный код: B3B7C1C4

💾 Сохраните эти данные!
```

## Примечания

- Пароль генерируется минимум из 9 символов (заглавные, строчные буквы, цифры, спецсимволы)
- Письмо с подтверждением ожидается до 5 минут (30 попыток по 10 секунд)
- Все уведомления приходят в указанный Telegram чат
- Для остановки процесса используйте Ctrl+C или команду /stop_auto

## Реферальный код

По умолчанию используется код: `B3B7C1C4`

Для изменения отредактируйте константу `REFERRAL_CODE` в файле main.py
